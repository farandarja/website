# **App Name**: Sirkus Splash

## Core Features:

- Homepage Showcase: Display hero banner with high-quality waterpark images, tagline, and CTA to purchase tickets.
- Online Ticket Purchase: Implement a simple online ticket purchasing system with options for ticket types (child, adult, family package) and date selection. Simulate payment integration and generate e-tickets with QR codes upon checkout.
- Virtual Tour: Create a virtual tour page with a photo gallery (grid/carousel layout), operational hours, address with Google Maps embed, and contact information (phone & WhatsApp).
- Facilities Showcase: Showcase facilities in a card layout with representative icons (e.g., gazebo, showers, lockers, food court, parking, prayer room).
- Attractions Listing: List waterpark attractions with large images and descriptions displayed on hover (e.g., spiral slides, wave pool, tipping bucket, kids pool, lazy river).
- News & Information: Implement a blog-style news and information section for promotions, events, and announcements, with a 'Read More' feature.
- AI Candid Photo Enhancement: The system should use generative AI tool that will increase resolution of user's candid photos, before storing and displaying on the candid gallery with masonry layout, zoom on click, and pinterest-style aesthetics.

## Style Guidelines:

- Primary color: Aqua (#7DF9FF) to evoke water and freshness.
- Background color: Light blue (#E0F7FA), a desaturated tint of the primary, for a soft and airy feel.
- Accent color: Yellow-orange (#FFB347) for a fun and energetic contrast in calls to action.
- Font: 'Poppins', sans-serif, known for its clean geometric forms which makes it very easy to read, it is suitable for all website text including body and headline.
- Use waterpark-themed icons for facilities and attractions.
- Employ a modern, responsive layout with plenty of whitespace and rounded corners.
- Incorporate light animations to make the website lively without affecting load times.
